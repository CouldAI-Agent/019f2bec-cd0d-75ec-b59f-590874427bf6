# Eris Electricity Tracker

A cross-platform dashboard and data entry application to track electricity consumption for Eris Life Sciences Ltd. 

## Features
- **Dashboard**: View Month-to-Date (MTD) and Year-to-Date (YTD) scoreboards for electrical consumption.
- **Consumption Charts**: Visual tracking of month-over-month power increase across various organizational areas and sub-areas/equipment.
- **Data Entry System**: Upload data files or manually input consumption readings.
- **AI Insights**: Generate AI-suggested insights on consumption trends to identify which areas or equipment are driving the increases.
- **Export & Reporting**: Links for Email, Excel, and PPT export capabilities.
- **Corporate Branding**: Clean, responsive layout using a blue and white color scheme.

## Tech Stack
- Flutter (iOS, Android, Web, Desktop support)
- `fl_chart` for dynamic dashboard visualizations

## Setup & Run
1. Ensure Flutter is installed and set up.
2. Clone the repository and run `flutter pub get` to install dependencies.
3. Run the application on your preferred platform:
   ```bash
   flutter run
   ```
